# Vrit Task

E-commerce product details page with image gallery and cart functionality.

## Technology Used
1. Next Js
2. React Js
3. Tailwind Css

## Installation
```
# Clone the repo
1. Clone this repository

# Move to the folder
2. cd vrit-task

# Install all the required dependencies
3. npm i

# Run the program
4. npm run dev
```
